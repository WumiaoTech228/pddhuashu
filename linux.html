14. Linux 专业系统安装
快速跳转发行版安装教程：
Win/Linux 双系统磁盘分区规划
安装双系统前，我们需要从您电脑已有的机械硬盘或固态硬盘中安全压缩出至少 50GB 的未分配空间，用于创建 Linux 的挂载根分区（/）与交换空间（Swap），这不会损坏您 Windows 原有的文件。
双系统引导安装向导说明
工程师将使用 Ventoy 引导为您加载主流 Linux 镜像（如 Ubuntu/Debian）。安装完成后，每次开机时都会出现 GRUB 双系统引导菜单，您可以自由选择进入 Windows 或 Linux 系统。
NVIDIA 独显驱动与 CUDA/CUDNN 安装
由于 Linux 系统自带的开源驱动（Nouveau）对 NVIDIA 显卡性能支持有限，我们将为您安装 NVIDIA 闭源驱动，并可根据您的开发需求配置 CUDA 与 cuDNN 深度学习计算环境。
国内软件源及输入法环境配置
系统安装完毕后，我们将为您配置国内的高速软件源（如清华源或阿里源），并为您安装配置好常用的搜狗拼音输入法/Fcitx5 以及基础编译开发环境（Git/GCC/Make）。
【Ubuntu 安装指南】 系统要求：CPU 双核 2GHz+，内存 ≥4GB (桌面)/≥512MB (服务器)，磁盘 ≥25GB，支持 UEFI/Legacy BIOS 引导。 一、Ubuntu Desktop (图形化桌面版) 1. 制作启动盘：从 ubuntu.com/download 下载 Desktop ISO，使用 Ventoy 写入 U 盘（推荐）或 Rufus（Windows 下选 DD 模式）。 2. 引导启动：插入 U 盘重启,按主板启动热键选择 U 盘，在 GRUB 菜单选择 "Try or Install Ubuntu" 进入 Live 桌面。 3. 启动安装：双击桌面 "Install Ubuntu" 图标，选择语言（中文简体）。 4. 键盘与网络：选择 "Chinese" 键盘布局。 建议先断网安装，否则安装器会在线拉取更新包导致安装极慢。 5. 更新与软件：选 "正常安装"（含浏览器、LibreOffice、媒体播放器）或 "最小安装"（仅浏览器+基础工具）。务必勾选 "安装第三方软件和硬件驱动"（自动解决 Wi-Fi/NVIDIA 显卡驱动）。 6. 安装类型与分区： - 全盘安装：「清除整个磁盘并安装 Ubuntu」→ 系统自动分区，会抹除磁盘所有数据。 - 双系统/自定义分区：选择「其他选项」(Something else)： · 点击空闲空间上的 "+" → EFI 系统分区：大小 500MB~1GB，用于 "EFI 系统分区"。 · 再点 "+" → Swap 交换空间：大小等同物理内存（≥8GB），用于 "交换空间"。 · 再点 "+" → 根分区 /：剩余所有空间，Ext4 日志文件系统，挂载点 "/"。 · 可选 /home 分区：单独划分用户数据分区。 7. 时区与用户：时区 "Shanghai"，输入用户名、计算机名、密码，选择「自动登录」或「需要密码登录」。 8. 写入安装：核对分区信息，点击「现在安装」→「继续」，等待进度条走完。提示「安装完成」时点击「现在重启」，黑屏时拔出 U 盘。 安装后配置： - 换国内源：打开「软件和更新」→「Ubuntu软件」→「下载自」→「其他站点」→ 选择 mirrors.tuna.tsinghua.edu.cn 或 mirrors.aliyun.com。 - 更新系统：终端运行 sudo apt update && sudo apt upgrade -y。 - 安装输入法：sudo apt install fcitx5 fcitx5-chinese-addons -y，重启后在「设置→区域与语言」添加「中文(智能拼音)」。 - 安装显卡驱动：打开「软件和更新」→「附加驱动」→ 选择 NVIDIA 专有驱动（推荐）→ 应用更改。 常见问题：若安装后无法进入 Windows，开机按 F12 选择 Windows Boot Manager；若 GRUB 不显示，进 BIOS 把 Ubuntu 设为第一启动项。 二、Ubuntu Server (命令行服务器版) 1. 引导启动：U 盘选择 Ubuntu Server 镜像，按 Enter 加载内核。 2. 语言选择：建议保持 "English" 以避免控制台中文乱码。 3. 键盘配置：Layout/Variant 选择 "Chinese / Chinese"。 4. 安装类型：选择 "Ubuntu Server" (标准) 或 "Ubuntu Server (minimized)" (最小化精简安装，更快更省资源)。 5. 网络配置：DHCP 自动获取 IP。如需静态 IP，手动编辑 Subnet（如 192.168.1.0/24）、Address（如 192.168.1.100）、Gateway（如 192.168.1.1）、DNS（如 223.5.5.5）。 6. 代理与镜像源：Proxy 留空按 Enter；镜像源地址改为 https://mirrors.tuna.tsinghua.edu.cn/ubuntu/ 加速下载。 7. 磁盘分区： - 新手选 "Use an entire disk" + 勾选 "Set up this disk as an LVM group"（LVM 逻辑卷，便于后期在线扩容）。 - 高级用户选 "Custom storage layout"： · /boot/efi：500MB, fat32（UEFI 引导）或 biosgrub：1MB（Legacy 引导）。 · /boot：1GB, ext4。 · LVM 卷组：创建 root-lv 挂载 /（ext4），swap-lv 作为交换空间。 8. 身份设置：输入姓名、服务器名(hostname)、用户名和高强度密码。 9. SSH 服务：务必勾选 "Install OpenSSH server"（否则安装后无法远程连接）。Import SSH Identity 可跳过。 10. 软件快照：可选安装 Docker、MicroK8s 等流行服务包，直接 Done 跳过。 11. 完成重启：等待系统写入完成，底部出现 "Reboot Now" 时回车重启并拔出 U 盘。 安装后配置： - 登录后查看 IP：ip addr show 或 hostname -I。 - 换国内源：sudo sed -i 's@//.*archive.ubuntu.com@//mirrors.tuna.tsinghua.edu.cn@g' /etc/apt/sources.list。 - 安装常用工具：sudo apt install build-essential git curl wget vim net-tools -y。 - 开启防火墙：sudo ufw enable && sudo ufw allow OpenSSH。 常见问题：若忘记勾选 SSH，可在服务器本地登录后执行 sudo apt install openssh-server -y 补救。
Debian Live (图形化) 与 Netinst (命令行) 安装教程
【Debian 安装指南】 系统要求：CPU 1GHz+，内存 ≥2GB (桌面)/≥256MB (服务器)，磁盘 ≥10GB (桌面)/≥4GB (服务器)，UEFI 或 Legacy BIOS。 一、Debian Live Desktop (图形化桌面版) 1. 下载与制作启动盘：从 debian.org/CD/live 下载对应桌面环境(GNOME/KDE/XFCE)的 Live ISO，使用 Ventoy 写入 U 盘。 2. 引导启动：U 盘启动后在 GRUB 菜单选择 "Debian Live with Desktop"，进入 Live 桌面体验环境。 3. 启动安装器：在桌面或应用菜单中双击打开 "Debian Installer" 图标。 4. 语言与时区：选「简体中文」，时区「Asia/Shanghai」，键盘布局「汉语」。 5. 分区选择： - 「抹除磁盘」：全盘安装，自动分区（数据清空）。 - 「手动分区」： · 新建 EFI 分区：大小 500MB~1GB，fat32，挂载 /boot/efi，标记 boot。 · 新建 Swap：等同内存大小（≥8GB），文件系统 linux-swap。 · 新建 / 根分区：剩余空间，ext4，挂载点 /，标记 root。 · 可选 /home：单独划分用户数据分区。 6. 用户创建：输入真实姓名、登录用户名、主机名和密码。建议留空 root 密码以默认使用 sudo。 7. 执行写入：确认分区概要无误后点击「安装」，等待文件复制完毕，点击重启。 安装后配置： - 换国内源：sudo sed -i 's/deb.debian.org/mirrors.tuna.tsinghua.edu.cn/g' /etc/apt/sources.list。 - 更新系统：sudo apt update && sudo apt upgrade -y。 - 安装中文输入法：sudo apt install fcitx5 fcitx5-chinese-addons -y。 - 安装常用软件：sudo apt install firefox-esr vim git curl build-essential -y。 常见问题：Debian 默认不包含非自由固件，若 Wi-Fi 无法使用，需下载包含 non-free firmware 的 ISO 版本；NVIDIA 显卡需手动启用 non-free 源后安装 nvidia-detect。 二、Debian Netinst (命令行服务器/网络精简安装) 1. 引导启动：U 盘选择 "Graphical install"（图形化向导）或 "Install"（纯文本），推荐 Graphical install。 2. 语言区域：Chinese (Simplified) → 中国 → 汉语。 3. 网络配置：DHCP 自动获取 IP。失败则手动填写 IP/掩码/网关/DNS。输入主机名，域名可留空。 4. 账户设置： - Root 密码：留空则禁用 root 直接登录，首个普通用户自动获得 sudo 权限（推荐）。 - 创建普通用户：全名、用户名、登录密码。 5. 磁盘分区： - 推荐「向导-使用整个磁盘」→ 分区方案选「将所有文件放在同一分区」(适合新手)。 - 确认「分区方案结束并将修改写入磁盘」→ 选「是」。 6. 包管理器配置： - 扫描额外安装介质 → 选「否」。 - 网络镜像国家 → 选「中国」。 - 镜像站点 → 选 mirrors.tuna.tsinghua.edu.cn 或 mirrors.aliyun.com。 - HTTP 代理 → 留空。 7. 软件选择（核心步骤）： - 需图形桌面：勾选 「Debian desktop environment」并在下方选 GNOME/KDE/XFCE 等桌面。 - 纯命令行服务器：取消勾选桌面环境。 - 务必勾选 「SSH server」和 「standard system utilities」。 8. GRUB 引导：选「是」将 GRUB 安装到主引导记录 → 选择硬盘设备（如 /dev/sda）。 9. 重启：选「继续」重启并拔出 U 盘。 安装后配置： - 查看 IP：ip addr 或 hostname -I。 - 换国内源：同上，编辑 /etc/apt/sources.list。 - 安装基础工具：sudo apt install sudo vim curl wget net-tools htop -y。 - 配置防火墙：sudo apt install ufw && sudo ufw enable && sudo ufw allow OpenSSH。 常见问题：Debian 默认 sudo 需要手动安装（若 root 留空会自动配置）；国内镜像连接失败可稍等重试或换用 mirrors.ustc.edu.cn。
Linux Mint 桌面版 (图形化) 安装与配置教程
【Linux Mint 安装指南】 系统要求：CPU 双核 2GHz+，内存 ≥2GB (推荐 4GB)，磁盘 ≥20GB，支持 UEFI/Legacy BIOS。Linux Mint 为纯桌面发行版，无独立服务器版。 1. 下载与制作启动盘：从 linuxmint.com/download 下载 Cinnamon/XFCE/MATE 版本 ISO，推荐 Cinnamon 旗舰版。使用 Ventoy 写入 U 盘。 2. 引导启动：U 盘启动后选择 "Start Linux Mint"，进入 Live 桌面先体验。 3. 启动安装：双击桌面的 "Install Linux Mint" 光盘图标。 4. 基础设置：选「中文(简体)」，键盘「Chinese」。注意 Mint 22 起改用新的安装器框架。 5. 多媒体解码器：务必勾选 「安装多媒体解码器」，自动安装 MP3/MP4/H.264 等媒体格式支持和微软字体包。 6. 磁盘分区： - 全盘安装：「擦除磁盘并安装 Linux Mint」→ 自动分区。 - 双系统/手动分区：选「其他选项」(Something else)： · EFI 分区：500MB~1GB，fat32，「EFI 系统分区」。 · Swap：等同内存大小（≥8GB），「交换空间」。 · / 根分区：剩余空间，ext4，挂载点 "/"。 · 可选 /home：单独划分用户数据分区。 7. 时区与用户：时区 "Shanghai"，输入用户名、计算机名、密码。 8. 写入与重启：点击「现在安装」→ 确认分区 → 等待进度条。完成后点击「现在重启」并拔 U 盘。 安装后配置（首次开机重点）： - 第一步：打开「欢迎屏幕」→「第一步」： · 运行「系统快照」(Timeshift) → 创建初始备份还原点。 · 运行「驱动管理器」→ 一键安装 NVIDIA 闭源驱动或博通 Wi-Fi 驱动。 · 运行「更新管理器」→ 更新所有核心系统组件。 - 换国内源：更新管理器 → 编辑 → 软件源 → 镜像 → 选择 mirrors.tuna.tsinghua.edu.cn 或 mirrors.aliyun.com。 - 安装输入法：系统自带 Fcitx5，在「输入法」设置中添加「中文拼音」。 - 安装常用软件：终端运行 sudo apt install git curl vim build-essential gimp -y。 常见问题：Mint 基于 Ubuntu，软件源与 Ubuntu 通用；若 Wi-Fi 不可用，先插网线进系统运行驱动管理器安装博通驱动。
Rocky Linux & AlmaLinux 桌面 (图形化) 与 服务器 (命令行) 安装教程
【Rocky Linux / AlmaLinux 安装指南】 系统要求：CPU 双核 2GHz+，内存 ≥2GB (桌面)/≥1GB (服务器)，磁盘 ≥20GB (桌面)/≥10GB (服务器)，UEFI/Legacy。两者 100% 兼容 RHEL，安装器均为 Anaconda。 一、图形化工作站安装 (Workstation GUI) 1. 下载与制作启动盘：从 rockylinux.org/download 或 almalinux.org/download 下载 DVD ISO，使用 Ventoy 写入 U 盘。 2. 引导启动：U 盘启动后选择 "Install Rocky Linux" 或 "Install AlmaLinux"。 3. 语言选择：「中文」→「简体中文(中国)」。 4. 软件选择： - 点击「软件选择」(Software Selection)，左侧选「带 GUI 的服务器」(Server with GUI) 或「工作站」(Workstation)。 - 右侧可勾选 「开发工具」「图形化管理工具」「办公套件」等附加包。 5. 安装目的地(分区)： - 点击「安装目的地」→ 选中目标硬盘。 - 选「自动」分区即可；或选「自定义」→「完成」进入分区界面 → 选 LVM → 点击「点击这里自动创建它们」→ 自动分配 /boot/efi、/boot、/、swap → 按需调大小 →「接受更改」。 6. 网络与主机名：点击「网络和主机名」，右上角开启网卡开关，左下角输入主机名 →「应用」。 7. 账户设置： - 点击「根密码」→ 设置 Root 密码。 - 点击「创建用户」→ 输入用户名+密码，勾选 「将此用户作为管理员」(授予 sudo 权限)。 8. 开始安装：所有 图标清除后点「开始安装」。安装完成后点「重启系统」，拔 U 盘。 安装后配置： - 首次开机需同意许可协议。 - 换国内源：sudo sed -e 's|^mirrorlist=|#mirrorlist=|g' -e 's|^#baseurl=http://dl.rockylinux.org|baseurl=https://mirrors.tuna.tsinghua.edu.cn/rocky|g' /etc/yum.repos.d/rocky*.repo (Rocky)；AlmaLinux 同理替换域名。 - 安装 EPEL：sudo dnf install epel-release -y。 - 安装开发工具：sudo dnf groupinstall "Development Tools" -y。 常见问题：RHEL 系默认防火墙开启，需放行服务端口：sudo firewall-cmd --permanent --add-service=ssh && sudo firewall-cmd --reload。 二、极简服务器安装 (Server Minimal CLI) 1. 引导与语言：同上，选「简体中文」。 2. 软件选择：选「最小安装」(Minimal Install)，无图形界面，开机纯命令行，资源占用最低。 3. 安装目的地：选中磁盘 →「自动」LVM 方案。 4. 网络配置：务必开启网卡开关 → 记录 IP 地址供 SSH 连接。 5. 账户与安装：设 Root 密码 → 可选创建管理员账户。点「开始安装」→ 重启拔 U 盘。 安装后配置： - 查看 IP：ip addr show 或 nmcli。 - 换国内源：同上编辑 .repo 文件。 - 安装基础包：sudo dnf install vim curl wget net-tools htop -y。 - 启动 Cockpit Web 管理：sudo dnf install cockpit && sudo systemctl enable --now cockpit.socket（访问 https://服务器IP:9090）。 常见问题：最小安装不含 man 手册和常用工具，需手动安装；SELinux 默认 enforcing，遇到权限问题可临时 setenforce 0 排查。
Fedora Workstation (图形化) 与 Server (命令行) 安装教程
【Fedora 安装指南】 系统要求：CPU 双核 2GHz+，内存 ≥4GB (Workstation)/≥2GB (Server)，磁盘 ≥20GB，UEFI 引导（Fedora 已全面转向仅 UEFI）。Fedora 是红帽技术先锋平台，Linus Torvalds 日常主力系统。 一、Fedora Workstation (图形化桌面版) 1. 下载与制作启动盘：从 fedoraproject.org/workstation/download 下载 ISO，使用 Ventoy 或 Fedora Media Writer 写入 U 盘。 2. 引导启动：U 盘启动选 "Start Fedora Workstation Live"，进入 Live GNOME 桌面。 3. 启动安装：弹出窗口点击 "Install to Hard Drive"（安装到硬盘）。 4. 语言键盘：选「中文(简体)」→「中文(中国)」。 5. 安装目的地(分区)： - 点击「安装目的地」→ 选中目标硬盘。 - 存储配置选「自动」(默认 Btrfs + 子卷方案，支持系统快照回滚) 或「自定义」(Blivet-GUI 高级分区器)。 - 若双系统选「自定义」→ 手动创建 /boot/efi (≥500MB, fat32)、/(≥20GB, btrfs/ext4)、swap。 6. 用户创建：点击「用户创建」→ 设置全名、用户名，勾选 「将此用户设为管理员」→ 输入密码。 7. 开始安装：点「开始安装」，完成后点「结束安装」→ 重启并拔 U 盘。 安装后配置： - 换国内源：sudo sed -e 's|^metalink=|#metalink=|g' -e 's|^#baseurl=http://download.example/pub/fedora/linux|baseurl=https://mirrors.tuna.tsinghua.edu.cn/fedora|g' /etc/yum.repos.d/fedora*.repo。 - 启用 RPM Fusion（第三方软件库）：sudo dnf install https://mirrors.rpmfusion.org/free/fedora/rpmfusion-free-release-$(rpm -E %fedora).noarch.rpm -y。 - 安装 NVIDIA 驱动：sudo dnf install akmod-nvidia -y（RPM Fusion 源提供）。 - 安装中文输入法：sudo dnf install fcitx5 fcitx5-chinese-addons -y。 - 安装常用软件：sudo dnf install vim git curl htop neofetch -y。 常见问题：Fedora 默认 Wayland + GNOME；若 NVIDIA 驱动异常，可在 GDM 登录界面齿轮图标切换 "GNOME on Xorg"。 二、Fedora Server (命令行服务器版) 1. 引导启动：U 盘选 "Install Fedora Server"。 2. 语言：选「中文(简体)」或 English。 3. 软件选择：选「Fedora Server 核心」(精简命令行) 或「标准 Fedora Server」(带基础管理工具+Cockpit)。 4. 网络与主机名：开启网卡 → 设主机名 → 记录 IP。 5. 磁盘分区：选目标硬盘 →「自动」LVM。 6. 管理员：设 Root 密码 + 创建管理员用户。 7. 执行安装：点「开始安装」→ 重启拔 U 盘。 安装后配置： - 换源同上。安装 Cockpit Web 管理：sudo dnf install cockpit -y && sudo systemctl enable --now cockpit.socket。 - 基础包：sudo dnf install vim curl wget net-tools -y。 常见问题：Fedora Server 默认使用 systemd-resolved 管理 DNS，修改 DNS 编辑 /etc/systemd/resolved.conf。
openSUSE Leap/Tumbleweed 图形化与命令行安装教程

Arch Linux 命令行手动与 archinstall 脚本自动化安装教程
【Arch Linux 安装指南】 系统要求：CPU x86_64，内存 ≥512MB (命令行)/≥2GB (桌面)，磁盘 ≥10GB。Arch 不提供图形安装器，所有操作在命令行完成。 一、archinstall 脚本自动化安装 (推荐，含桌面/命令行) 1. 下载与制作启动盘：从 archlinux.org/download 下载 ISO，用 Ventoy 或 dd 写入 U 盘（ 不要用 Rufus ISO 模式，用 DD 模式）。 2. 引导与联网：U 盘启动进入 Live 命令行。台式机插网线自动联网；笔记本输入 iwctl → device list → station wlan0 scan → station wlan0 get-networks → station wlan0 connect SSID 连 Wi-Fi。用 ping archlinux.org 验证联网。 3. 启动安装脚本：输入 archinstall 回车，进入交互式 TUI 菜单。 4. 配置菜单： - "Language"：建议 "English"（防乱码）。 - "Mirrors" → "Mirror region"：选 "China"（启用国内镜像加速）。 - "Disk configuration" → "Layout" → 选目标硬盘 → "Wipe all selected drives" → 文件系统推荐 btrfs（支持快照）或 ext4。 - "Loader"：选 systemd-boot（推荐，简洁）或 grub（更通用）。 - "Profile"： · 装桌面：Type → desktop → 选 GNOME/KDE/XFCE → 显卡驱动（NVIDIA 选 nvidia，Intel/AMD 选开源）。 · 装服务器：Type → minimal（纯命令行）。 - "Audio"：选 pipewire。 - "Kernels"：linux（最新稳定内核）或 linux-lts（长期支持）。 - "Create user"：用户名+密码，勾选 "superuser" 赋予 sudo 权限。 - "Root password"：设置 root 密码。 - "Network configuration"：选 "Use NetworkManager"（ 非常重要！）。 - "Timezone"："Asia/Shanghai"。 - "Optional repositories"：勾选 multilib（32位兼容库）。 5. 执行安装：点 "Install" → 系统下载包并安装（约 10-30 分钟视网速）。完成后选 "no" 不进入 chroot → 输入 reboot 重启拔 U 盘。 安装后配置： - 开机后用普通用户登录（root 默认禁止 SSH）。 - 换国内源：sudo sed -i '1i Server = https://mirrors.tuna.tsinghua.edu.cn/archlinux/\$repo/os/\$arch' /etc/pacman.d/mirrorlist。 - 安装 AUR 助手：sudo pacman -S --needed git base-devel && git clone https://aur.archlinux.org/yay-bin.git && cd yay-bin && makepkg -si。 - 安装输入法：sudo pacman -S fcitx5 fcitx5-chinese-addons fcitx5-gtk fcitx5-qt。 常见问题：安装时若下载失败，先更新密钥环 pacman -Sy archlinux-keyring；archinstall 失败可重试，注意选择正确的磁盘别误删数据。 二、传统命令行手动安装 (纯命令模式) 1. 联网：同上用 iwctl 或插网线。更新系统时钟：timedatectl set-ntp true。 2. 分区磁盘：cfdisk /dev/sda → 创建 EFI 分区(500M, type: EFI System)、Swap(8G, type: Linux swap)、根分区(剩余, type: Linux filesystem)。 3. 格式化：mkfs.vfat -F32 /dev/sda1（EFI）、mkswap /dev/sda2 && swapon /dev/sda2（Swap）、mkfs.ext4 /dev/sda3（根）。 4. 挂载分区：mount /dev/sda3 /mnt → mkdir -p /mnt/boot → mount /dev/sda1 /mnt/boot。 5. 换源并安装基础系统：编辑 /etc/pacman.d/mirrorlist 把 China 源移到最前面 → pacstrap -K /mnt base linux linux-firmware base-devel NetworkManager vim sudo。 6. 生成 fstab：genfstab -U /mnt >> /mnt/etc/fstab。 7. chroot 进入新系统：arch-chroot /mnt。 8. 系统初始化： - 时区：ln -sf /usr/share/zoneinfo/Asia/Shanghai /etc/localtime && hwclock --systohc。 - 本地化：编辑 /etc/locale.gen 取消注释 en_US.UTF-8 和 zh_CN.UTF-8 → 运行 locale-gen → echo "LANG=en_US.UTF-8" > /etc/locale.conf。 - 主机名：echo "myarch" > /etc/hostname。 - 用户与密码：passwd 设 root 密码 → useradd -m -G wheel 用户名 && passwd 用户名 → EDITOR=vim visudo 取消注释 %wheel ALL=(ALL:ALL) ALL。 9. 安装 GRUB：pacman -S grub efibootmgr → grub-install --target=x86_64-efi --efi-directory=/boot --bootloader-id=GRUB → grub-mkconfig -o /boot/grub/grub.cfg。 10. 启用联网：systemctl enable NetworkManager。 11. 退出重启：exit → umount -R /mnt → reboot 拔 U 盘。
