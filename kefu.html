02. 数据备份与安全警告
C盘数据备份警示
重要提示：重装系统会清空C盘的所有数据，包括桌面文件、文档、下载目录以及微信/QQ的聊天记录。请务必将重要数据备份到其他分区中！
多硬盘/多分区安全告知
请确认您的电脑上除了C盘外是否还有其他盘（如D盘、E盘）？如果有，重装系统默认只格式化C盘，其他盘的文件均会安全保留，不受影响。
浏览器书签与账号备份提示
如果您有在浏览器中保存的重要书签、账号密码或行业软件本地数据，请提前导出备份。系统重装后，原有软件需重新安装，浏览器数据可登录账号进行云同步。
磁盘 BitLocker 加密警告
重要安全提示：重装前请确认您的 C 盘或 D 盘等分区是否开启了 BitLocker 磁盘加密（如果开启，盘符上会有个锁的图标）。若开启了，请务必先进入控制面板将其彻底解密关闭，或者导出保存好您的恢复密钥，以防重装后其他盘被死锁无法读取。

07. 重启引导与自动安装
重启进入自动重装环境
重装软件环境已经配置完毕，现在需要重启电脑。重启后电脑会自动显示引导菜单并进入重装软件的PE自动安装环境，您无需敲击任何按键，网络及声音自动部署，耐心等待其自动执行格式化和写入即可。
重启进不去BIOS/启动菜单解决方法
如果重启后没有自动进入重装环境，而是直接回到了原系统或提示引导失败，说明您的系统开启了引导保护。请先手动关机，开机时连击 Delete 或 F2 进入 BIOS，在安全或启动选项中检查引导顺序。
安全启动报错解决方法
如果重启进入重装软件环境时屏幕报错或提示安全启动违规（Secure Boot Violation），需要重新开机进BIOS，在安全设置（Security）里将Secure Boot（安全启动）设置为Disabled（关闭），保存重启后再试。
老旧电脑 Legacy 与 UEFI 引导模式切换
如果重启后找不到重装软件的引导项，可能是老旧主板未开启 UEFI 兼容。请进入 BIOS 设置，在 Boot 或 Startup 菜单中，将 Boot Mode 从 Legacy 修改为 UEFI（或者将 CSM 兼容支持模块开启），保存并重启后再试。
08. 安装中遇到特殊问题解决
看不到电脑硬盘（Intel VMD限制）
如果重装界面找不到任何磁盘，这是因为您的电脑是Intel 11代及以上的处理器，默认开启了Intel VMD硬盘技术。我们需要远程为您加载相应的IRST驱动，或者需要您进入BIOS关闭VMD设置。
引导格式不符（GPT与MBR硬盘转换）
如果重装软件提示无法安装到当前磁盘，因为硬盘分区表是 MBR 或 GPT 格式不符，这是因为您的系统分区表类型与当前的启动模式冲突。工程师需要将该磁盘重新初始化并转换分区表格式，请确认。
重装后其他盘符（D盘/E盘）不见了或显示脱机
如果进入桌面后发现 D 盘或 E 盘等其他分区不见了，请右键点击桌面“此电脑”选择“管理”，进入“磁盘管理”。如果看到其他硬盘状态为“脱机”，只需在其上点击右键选择“联机”；如果显示未分配盘符，右键为其“更改驱动器号和路径”分配盘符即可恢复。

13. 电脑系统疑难杂症解决
蓝屏崩溃日志分析
请您在 C:\Windows\Minidump 路径下查找是否有后缀为 .dmp 的蓝屏日志文件并发送给工程师。我们将使用专业分析工具定位是哪个驱动程序或硬件硬件导致的蓝屏崩溃。
系统运行库 DLL 报错修复
针对您运行软件或游戏时提示缺少 msvcp140.dll、vcruntime140.dll 或 DirectX 3D 组件的问题，工程师将为您重新部署全套微软 Visual C++ 2005-2022 最新运行库及 DirectX 修复组件。
显卡驱动冲突黑屏/闪烁解决
我们会首先引导电脑进入安全模式，使用 DDU (Display Driver Uninstaller) 工具彻底清理残留的旧显卡驱动和注册表，随后为您重新纯净安装最稳定的显卡官方驱动，解决游戏闪退或黑屏。
网络丢包延迟与网卡修复
针对您的网络访问异常或游戏高延迟丢包，我们将为您进行网卡驱动重置、清空 DNS 缓存并配置最优公共 DNS（如阿里或腾讯DNS），并修复 LSP 套接字以恢复正常网络。
14. Linux 专业系统安装
Win/Linux 双系统磁盘分区规划
安装双系统前，我们需要从您电脑已有的机械硬盘或固态硬盘中安全压缩出至少 50GB 的未分配空间，用于创建 Linux 的挂载根分区（/）与交换空间（Swap），这不会损坏您 Windows 原有的文件。
双系统引导安装向导说明
工程师将使用 Ventoy 引导为您加载主流 Linux 镜像（如 Ubuntu/Debian）。安装完成后，每次开机时都会出现 GRUB 双系统引导菜单，您可以自由选择进入 Windows 或 Linux 系统。
NVIDIA 独显驱动与 CUDA/CUDNN 安装
由于 Linux 系统自带的开源驱动（Nouveau）对 NVIDIA 显卡性能支持有限，我们将为您安装 NVIDIA 官方闭源驱动，并可根据您的开发需求配置 CUDA 与 cuDNN 深度学习计算环境。
国内软件源及输入法环境配置
系统安装完毕后，我们将为您配置国内的高速软件源（如清华源或阿里源），并为您安装配置好常用的搜狗拼音输入法/Fcitx5 以及基础编译开发环境（Git/GCC/Make）。
15. Steam 安装与使用优化
Steam 官方纯净客户端安装
很多搜索引擎下载的 Steam 带有病毒或捆绑流氓软件。我们为您下载和安装的均是 Valve 官方原版 Steam 客户端，确保纯净安全无广告干扰。
Steam 社区及商店无法访问解决 (-118错误)
由于国内网络限制，Steam 商店和社区经常会出现错误代码 -118（网络连接超时）。我们将为您配置免费的 Steam 社区网络加速器或修改 Hosts 解析，保障商店与社区顺畅打开。
Steam 游戏库路径导入与备份
如果您之前下载过游戏，请不要重复下载。工程师可以通过设置 Steam 的【库文件夹】指向您原有的游戏目录，快速扫描并识别已有的游戏文件，为您节省大量下载时间。
Steam 账号注册安全与防抢注说明
我们将引导您在官方渠道注册 Steam 账号，帮您绑定邮箱并启用手机 Steam Guard 令牌双重验证，有效防止账号被盗或被他人恶意抢注注册。
